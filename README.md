 # Smart Expense Tracker

A modern Android Expense Tracking Application built with Kotlin and Jetpack Compose.

## Features

- User Authentication
- Add Income & Expenses
- Expense Categories
- Dashboard Overview
- Financial Reports
- Modern Material Design UI
- Dark Mode Support
- Local Data Storage

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/smart-expense-tracker.git
```

2. Open project in Android Studio

3. Sync Gradle

4. Run the application
